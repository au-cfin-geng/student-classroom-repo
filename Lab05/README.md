# Lab05 — Tool / MCP-Aware Workflow

**Lab question:** How do I safely connect Claude to files, tools, papers, and research context?

## Where to work

Do all your lab work inside `Lab05/work/`.

## What to do

1. Open the course dashboard (`make dashboard`)
2. Select Lab05 in the sidebar — read the lesson
3. Copy `Lab05/do_lab_prompt.md` into Claude Code
4. Claude writes output files into `Lab05/work/`
5. Read and evaluate what Claude produced
6. Commit your work
7. Run the Finish Lab prompt → `Lab05Submission/`

## How to finish

Copy `lecture-repo/prompts/finish_lab.md` into Claude Code.
Claude packages your work from `Lab05/work/` into `Lab05Submission/`.
