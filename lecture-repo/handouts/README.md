# Handouts — Agentic Clinical Research Studio

Student-facing teaching documents for each lab. Each handout follows the same 10-section structure:

1. Opening Story — the research problem in narrative form
2. The Old Workflow — how researchers currently handle this
3. The Agentic Workflow — how the lab concepts change it
4. Core Concept — plain-language explanation for medical researchers
5. Clinical Example — concrete application to the brain MRI case study
6. In-Class Activity — structured 15-20 minute group exercise
7. Artifact Contract — what files the lab produces and what counts as complete
8. Common Failure Modes — what typically goes wrong and how to avoid it
9. Responsible Use — governance, ethics, and scientific integrity considerations
10. Further Learning — curated search terms, stable links, and resources to go deeper
