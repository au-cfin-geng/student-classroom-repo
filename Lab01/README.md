# Lab01 — 

**Lab question:** 

## Where to work

Do all your lab work inside `Lab01/work/`.

## What to do

1. Open the course dashboard (`make dashboard`)
2. Select Lab01 in the sidebar — read the lesson
3. Copy `Lab01/do_lab_prompt.md` into Claude Code
4. Claude writes output files into `Lab01/work/`
5. Read and evaluate what Claude produced
6. Commit your work
7. Run the Finish Lab prompt → `Lab01Submission/`

## How to finish

Copy `lecture-repo/prompts/finish_lab.md` into Claude Code.
Claude packages your work from `Lab01/work/` into `Lab01Submission/`.
