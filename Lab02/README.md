# Lab02 — Prompt Contracts

**Lab question:** How do I turn a vague clinical research request into a verifiable task?

## Where to work

Do all your lab work inside `Lab02/work/`.

## What to do

1. Open the course dashboard (`make dashboard`)
2. Select Lab02 in the sidebar — read the lesson
3. Copy `Lab02/do_lab_prompt.md` into Claude Code
4. Claude writes output files into `Lab02/work/`
5. Read and evaluate what Claude produced
6. Commit your work
7. Run the Finish Lab prompt → `Lab02Submission/`

## How to finish

Copy `lecture-repo/prompts/finish_lab.md` into Claude Code.
Claude packages your work from `Lab02/work/` into `Lab02Submission/`.
