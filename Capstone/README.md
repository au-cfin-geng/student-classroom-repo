# Capstone — Mini-Project

**What this lab teaches:** Applying the full agentic workflow to your own clinical research problem. You define the question, design the workflow, produce the artifacts, and present to a non-ML audience.

## Where to work

Do all your capstone work inside `Capstone/work/`.

## What to do

1. Complete at least Lab00–Lab03 before the capstone
2. Open the course dashboard — select the Capstone
3. Define your own clinical research question
4. Design a workflow using at least 3 course concepts
5. Run your workflow, produce artifacts in `Capstone/work/`
6. Commit your work
7. Run the Finish Lab prompt → `CapstoneSubmission/`

## How to finish

Copy `lecture-repo/prompts/finish_lab.md` into Claude Code.
Claude packages your work from `Capstone/work/` into `CapstoneSubmission/`.
