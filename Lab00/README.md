# Lab00 — Welcome to Agentic Clinical Research

**What this lab teaches:** The agentic research mindset — Claude as a systematic, traceable collaborator, not a search box. Every significant action produces a committed artifact.

## Where to work

Do all your lab work inside `Lab00/work/`.

## What to do

1. Open the course dashboard (`make dashboard`)
2. Read the L00 lesson in the sidebar
3. Copy `Lab00/do_lab_prompt.md` into Claude Code
4. Claude creates your first orientation artifact in `Lab00/work/`
5. Personalise the orientation with your real research question
6. Commit your work
7. Run the Finish Lab prompt → `Lab00Submission/`

## How to finish

Copy `lecture-repo/prompts/finish_lab.md` into Claude Code.
Claude packages your work from `Lab00/work/` into `Lab00Submission/`.
